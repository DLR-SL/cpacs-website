CPACS Schema

Permalink: https://www.cpacs.de/schema/v3_5_0/cpacs_schema.xsd

Copy the following code into your XML file to link to the corresponding CPACS schema: 

<?xml version="1.0" encoding="UTF-8"?>
<cpacs xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:noNamespaceSchemaLocation="https://www.cpacs.de/schema/v3_5_0/cpacs_schema.xsd">
