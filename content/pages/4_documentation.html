Example files

Some examples can be downloaded from GitHub. If you have an interesting CPACS file, for example from a project work, you would like to share in order to support other users we would be more than happy to make it available to the community by adding it to the example folder.

Project specific versions

Within projects it is often necessary to test extensions and adaptations of the schema in practice before implementing them in an official release. Therefore, a temporary XML Schema Document (XSD) is created on the basis of an official release. If the changes made to the XSD are considered successful after the end of the project, they are then incorporated into an official release.
