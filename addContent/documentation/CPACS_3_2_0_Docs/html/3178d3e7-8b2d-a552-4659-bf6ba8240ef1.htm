Remarks

0. General overview

This type contains the reference values used for normalizing local aerodynamic coefficients. The reference point is defined in absolut CPACS-coordinates.

1. <Flightshapepoint> (mandatory)

Flight shape point, specified in absolut CPACS-coordinates The reference point is used as origin for calculating the local moment coefficients. See the documentation of the global CPACS-Element for further information.

2. <Jigshapepoint> (mandatory)

Jig shape point, specified in absolut CPACS-coordinates. The corredponding point on the initial CPACS geometry.

3. <area> (mandatory)

Local reference area.

4. <length> (mandatory)

Local reference length, used for normalizing all the moment coefficients. The length is always given in the middle of the wing segment.
