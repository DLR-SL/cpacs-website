Remarks

0. General overview

This type contains the integral aerodynamic coefficients, as well as spanwise coefficient distributions for one fuselage.

1. <fuselageUID> (mandatory)

Reference to the fuselage

2. <coefficients> (mandatory)

Total force and moment coefficients of a fuselage.

3. <segments> (mandatory)

A list of segments with their spanwise coefficient distributions.
