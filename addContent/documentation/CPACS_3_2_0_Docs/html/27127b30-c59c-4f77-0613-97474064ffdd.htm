Remarks

0. General overview

This type contains the integral aerodynamic coefficients, as well as spanwise coefficient distributions for one wing.

1. <wingUID> (mandatory)

Reference to the wing

2. <coefficients> (mandatory)

Total force and moment coefficients of a wing.

3. <segments> (mandatory)

A list of segments with their spanwise coefficient distributions.
