Remarks
hingeMomentsMap type, containing a hinge moments map with aerodynamic data. Array order is: angleOfAttack min->max then angleOfSideslip then reynoldsNumber then machNumber.All coefficients in the aeroperformanceMap relate to the CPACS coordinate system. See documentation of the CPACS-Element for further information.
