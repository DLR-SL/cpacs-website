Remarks

0. General overview

This type contains the aerodynamic coefficients of one fuselage segment.

1. <segmentUID> (mandatory)

A reference to the fuselage segment.

2. <strip> (mandatory)

A list of nodes, each containing the local aerodynamic coefficients of a fuselage segment's strip.
