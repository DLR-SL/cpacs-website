Remarks
ControlSurfacePerformanceMap type, containing a delta performance map with aerodynamic data for a control surface. Array order is: relativeDeflection min->max then angleOfAttack then angleOfSideslip then altitude then machNumber. AngleOfAttack, angleOfSideslip, altitude and machNumber are taken from the basic performance map one level above.
