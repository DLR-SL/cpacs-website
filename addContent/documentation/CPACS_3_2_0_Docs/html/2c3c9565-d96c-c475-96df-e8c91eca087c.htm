Coordinate System(s)

Euler angles

  • Attitude of body fix to earth fix coord.sys.
  • ψ psi – heading
  • θ theta – pitch angle
  • Φ phi – bank angle
  • Rotation order: ψ, θ, Φ

Reference of attitude values

  • x,y,z (vector to ref point in EF)
  • dx,dy,dz (vector in EF)
  • u,v,w (vector in refCoord) (body fix)
  • du,dv,dw (vector in refCoordUid) (body fix)

(dx,dy,dy --> Euler transformation (psi, theta, phi) --> u,v,w)
