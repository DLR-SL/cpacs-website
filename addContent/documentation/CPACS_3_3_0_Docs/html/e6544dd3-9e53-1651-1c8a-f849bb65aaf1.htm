Remarks

The vehiclesType contains all vehicle-specific data.

This includes the vehicle itself (i.e. aircraft and rotorcraft). Furhtermore, components (e.g. engines, structuralElements, etc.) as well as physical properties of materials and fuels can be predefined for easy and consistent reuse via uID-references.
