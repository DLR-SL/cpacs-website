Remarks

[WARNING: This type is known to be susceptible to inconsistencies and might therefore be removed in a future version of CPACS]

The geometry of the cabin roughly corresponds to the available design space in the cabin. It is given in terms of constant height contour lines. The lines all share a common x-vector. The y vector provides the lateral contour at Z-coordinate provided by the constant value z. One or more contour lines can be given. The cabin geometry is assumed to be symmetric.
