Remarks
A spoiler is defined via its outerShape relative to the componentSegment. The WingCutOut defines the area of the upper skin that is removed by the spoiler. Structure is similar to the wing structure. The mechanical links between the spoiler and the parent are defined in tracks. The deflection path is described in path. Additional actuators, that are not included into a track, can be defined in actuators.
