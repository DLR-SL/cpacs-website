Remarks
controlSurfaceHingeMomentMap type, containing a moment map with aerodynamic data for a control surface. Array order is: controlParameters min->max then angleOfAttack then angleOfSideslip then reynoldsNumber then machNumber. AngleOfAttack, angleOfSideslip, reynoldsNumber and machNumber are taken from the basic performance map one level above.
